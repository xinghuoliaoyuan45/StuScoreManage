大三下学期课程设计
老师要求hsqldb数据库 使用java   

- 用图形或命令行菜单方式打开增、删、改、模糊查询、浏览功能界面；
- 学生数据表的字段要求至少包含：学号、姓名、班级、三门以上课程；
- 能按学期、按班级完成对学生成绩的录入、修改；
- 能按班级统计学生的成绩，求学生的总分及平均分，并能根据学生的平均	成绩进行排序；
- 能查询学生成绩，不及格科目及学生名单；
- 能按班级输出学生的成绩单（excel或其它格式）；
- 导入、导出。
